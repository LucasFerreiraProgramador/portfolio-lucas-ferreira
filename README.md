## 🛠️ Stack Tecnológico

### Frontend
- **React 18.3**: Biblioteca JavaScript para construção de interfaces
- **TypeScript 5.8**: Tipagem estática para código mais seguro e manutenível
- **Vite 5.4**: Build tool ultrarrápido para desenvolvimento e produção
- **Tailwind CSS 3.4**: Framework CSS utility-first para estilização

### UI & Componentes
- **Shadcn/ui**: Componentes de alta qualidade e acessibilidade
- **Radix UI**: Primitivos de componentes sem estilo
- **Lucide React**: Ícones SVG consistentes e escaláveis
- **Framer Motion 12.25**: Animações fluidas e intuitivas

### Funcionalidades Avançadas
- **React Query 5.83**: Gerenciamento de estado e cache de dados
- **React Router 6.30**: Roteamento client-side
- **Recharts 2.15**: Gráficos e visualizações de dados
- **Supabase**: Backend e autenticação
- **tsParticles 3.0**: Sistema de partículas para efeitos visuais

### Desenvolvimento
- **ESLint 9.32**: Linting de código
- **Autoprefixer 10.4**: Prefixos CSS automáticos
- **PostCSS 8.5**: Processamento de CSS

## 🚀 Como Executar Localmente

### Pré-requisitos

- Node.js 18+ instalado
- npm, yarn ou pnpm como gerenciador de pacotes
- Git para clonar o repositório

### Instalação

```bash
# Clone o repositório
git clone https://github.com/LucasFerreiraProgramador/portfolio-lucas-ferreira

# Navegue até o diretório
cd cyber-portfolio

# Instale as dependências
npm install
# ou
pnpm install
```

### Desenvolvimento

```bash
# Inicie o servidor de desenvolvimento com hot reload
npm run dev
# ou
pnpm dev
```

O projeto estará disponível em `http://localhost:5173`

### Build para Produção

```bash
# Compile para produção
npm run build
# ou
pnpm build

# Visualize o build localmente
npm run preview
# ou
pnpm preview
```

## 🎨 Design System

### Paleta de Cores

| Cor | Hex | Uso |
|-----|-----|-----|
| Fundo Primário | `#0b0e14` | Background principal |
| Neon Ciano | `#00d9ff` | Destaque e borders |
| Neon Rosa | `#ff006e` | Acentos e hover |
| Branco | `#ffffff` | Texto principal |
| Cinza | `#8892b0` | Texto secundário |

### Tipografia

- **Display**: Títulos e headings (font-weight: 700-900)

- **Botões**: Efeito neon com transições suaves
- **Cards**: Glassmorphism com backdrop blur
- **Inputs**: Estilo minimalista com focus states

## 🚀 Deploy

O portfólio está hospedado em múltiplas plataformas:

- **Recomendada**: Vercel 
- **Alternativa**: Netlify
- **CDN**: Cloudflare

### Deploy no Vercel

```bash
# Instale o Vercel CLI
npm install -g vercel

# Faça deploy
vercel
```

### Deploy no Netlify

```bash
# Instale o Netlify CLI
npm install -g netlify-cli

# Faça deploy
netlify deploy --prod --dir=dist
```
